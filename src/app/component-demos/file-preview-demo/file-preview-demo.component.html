The File Preview Component allows you to display uploaded files as thumbnails with optional overlay preview, download, and delete actions.


Supported File Types

  • Images — JPG, JPEG, PNG, GIF, WEBP, BMP, SVG, ICO (thumbnail generated, full overlay preview)
  • PDF — thumbnail and inline overlay preview
  • DOCX — thumbnail and inline overlay preview
  • EXCEL — thumbnail and inline overlay preview
  • Other types — displayed with a generic file icon; no preview available

Behavior

  • Renders image thumbnails automatically from a URL, Blob, File, ArrayBuffer, or Base64 input
  • Opens a full-screen overlay for images, PDFs, DOCX and Excel workbooks when the thumbnail is clicked (inline rendering in supported formats)
  • Falls back to a file-type icon for unsupported formats
  • Action bar below each thumbnail shows configurable preview, download, and delete buttons
  • Supports live updates via an itemsStream Observable

Import the FilePreviewComponent in your component and use it via <mad-file-preview> .
