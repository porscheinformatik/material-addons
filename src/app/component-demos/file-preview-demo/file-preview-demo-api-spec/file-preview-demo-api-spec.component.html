Properties

Property Type Default Value Description
items FilePreviewItem[] | null null Array of file items to display. Use with Angular change detection for live updates.
config FilePreviewConfig see below Configuration object controlling thumbnail size and action visibility.
labels FilePreviewLabels undefined Optional object to override default ARIA labels and user-visible strings.

FilePreviewConfig Properties

Property Type Default Value Description
thumbnailSize 'sm' | 'md' | 'lg' | { width: number; height: number } 'md' Thumbnail dimensions. Presets: sm = 96x96, md = 132x132, lg = 180x180 px.
showOverlayPreview boolean true When false, thumbnail clicks and the preview button are disabled.
showActionIcons boolean true Shows the action bar below each thumbnail.
showDeleteAction boolean true Adds a Delete button to the action bar and overlay header.
showPreviewAction boolean true Adds a Preview icon button to action bars when overlay preview is enabled.
showDownloadAction boolean true Adds a Download icon button to action bars and overlay header.
actions FilePreviewAction[] [] Additional custom action buttons appended to every action bar.
generatePdfThumbnails boolean false When true, each PDF is fully downloaded and rasterised to a JPEG thumbnail on load. Requires pdfjs-dist to be installed. Disabled by default to avoid unexpected network cost on large lists.
excelPreviewRowLimit number 200 Maximum number of rows rendered in the Excel preview overlay (excluding the header row). A "Showing first N of M rows" notice is appended when the sheet exceeds this limit. Set to Infinity to render all rows.

FilePreviewItem Properties

Property Type Default Value Description
id string required Unique stable identifier used for trackBy and delete events.
name string required Display name including extension, e.g. report.pdf.
mimeType MimeType | string undefined MIME type for accurate kind detection. Use MimeType union type for autocomplete of supported types (images, PDF, DOCX, Excel, etc.). Falls back to file extension in name.
source string | FilePreviewBase64Input | Blob | File | ArrayBuffer undefined File content. Accepts URL, data URI, structured Base64, Blob, File, or ArrayBuffer.
size number undefined File size in bytes. Displayed in the overlay header.
meta Record<string, unknown> undefined Additional metadata passed through to all output events unchanged.

FilePreviewBase64Input Properties

Property Type Default Value Description
data string required Raw Base64 string or fully qualified data URI (data:<mime>;base64,<data>).
mimeType MimeType | string required MIME type of the Base64 data. Use MimeType union type for autocomplete of supported types.

Supported Preview Types

Kind Preview Notes
Images (JPG, PNG, GIF, WEBP, SVG, BMP, ICO) Thumbnail + overlay Rendered from Blob/File/URL/Base64
PDF Thumbnail + inline overlay Inline view via browser PDF object. Rasterised thumbnail requires pdfjs-dist and [config]="{ generatePdfThumbnails: true }". Worker URL configurable via PDF_WORKER_SRC token.
DOCX Thumbnail + inline overlay Rendered using docx-preview when available
Excel / CSV / ODS Thumbnail + inline preview Rendered using @e965/xlsx when available. Row cap configurable via excelPreviewRowLimit.
Other Icon Generic file icon; no inline preview

Events

Event Description
previewOpened Emits ResolvedFilePreviewItem when the overlay preview is opened.
actionClicked Emits { action, item } when a custom action button is clicked.
deleteClicked Emits ResolvedFilePreviewItem when the Delete button is clicked. The host is responsible for removing the item.
