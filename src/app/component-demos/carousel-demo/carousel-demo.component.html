Import the CarouselComponent in your app.module.ts or component and use it via <mad-carousel>

Each slide must apply the CarouselSlideDirective, this is necessary to apply all needed styles. Import CarouselSlideDirective in your app.module.ts or component and use it via <div madCarouselSlide>

Using the CarouselShortTextDirective the user can provide a short text that will be displayed inside the navigation dots of the carousel. If one of the slides provides a value for this directive, then all slides need to provide a value.

To use the directive, import CarouselShortTextDirective in your app.module.ts or component and use it via <div madCarouselShortText="short text"> or <div [madCarouselShortText]="shortTextVariable">
