Use to display breadcrumb navigation that helps users understand their current location within the application hierarchy, navigate back to previous levels, and optionally trigger a copy action.

Use When

  • Showing the current location within a page hierarchy
  • Allowing users to navigate back to parent pages
  • Improving orientation in multi-level navigation structures
  • Providing an optional action to copy the current breadcrumb context

Behavior

  • Displays a list of breadcrumb items in hierarchical order
  • Supports clickable breadcrumb items using links
  • Allows the current page to be displayed as a non-clickable item
  • Accepts an array of breadcrumb items with labels and optional navigation targets
  • Shows a copy action by default, which can be hidden if needed
  • Emits an event when the copy action is triggered

Import the BreadcrumbComponent in your app.module.ts and use it via <mad-breadcrumb> .
