Properties

Property Type Default Value Description
breadcrumbs BreadcrumbItem[] - Required. Defines the breadcrumb items to display.
showCopy boolean false (Optional) Controls whether the copy action is shown. Default is false.

BreadcrumbItem

Field Type Description
label string The text displayed for the breadcrumb item.
route string[] (Optional) Router navigation target for the breadcrumb item.
href string (Optional) Link target for the breadcrumb item.

Events

Event Description
copy Emits when the copy action is triggered.

CSS Custom Properties

Css Custom Property Description
--breadcrumb-separator-color Separator and current breadcrumb color
