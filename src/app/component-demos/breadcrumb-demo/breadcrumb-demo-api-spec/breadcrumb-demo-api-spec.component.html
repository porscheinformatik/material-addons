Properties

Property Type Default Value Description
breadcrumbs BreadcrumbItem[] required Breadcrumb items rendered in hierarchy order.
showCopy boolean false Controls whether the copy action button is shown.
title string 'Copy' Tooltip text for the copy action button.

BreadcrumbItem

Field Type Description
label string Text displayed for the breadcrumb item.
route string[] Optional Angular router target. When present, the item renders as a router link.
href string Optional external link target. Used when no router target is provided.

Events

Event Description
copy Emits when the user activates the copy action button.

CSS Custom Properties

CSS Custom Property Description
--breadcrumb-separator-color Color used for separators and the current breadcrumb item.
