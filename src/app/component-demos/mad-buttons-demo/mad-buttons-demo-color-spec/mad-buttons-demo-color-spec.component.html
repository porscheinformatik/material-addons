Variant Colors

Variant Token Usage
Primary filled --main-primary Background for mad-primary-button and primary filled Material buttons.
Primary outlined and text --main-primary Label, border, and icon color for primary outline, link, and icon buttons.
Accent outlined and text --mat-sys-secondary Label and border color for accent buttons using color="accent".
Danger and warn --error-color Background for mad-danger-button and label/border color for color="warn".
Disabled opacity --mad-button-disabled-opacity Disabled buttons keep their variant color identity and reduce opacity through theme token color mixing.

CSS Custom Properties

Css Custom Property Description
--main-primary Primary button, outline, text, icon, and group color.
--error-color Danger button background and warn outline/text color.
--mat-sys-secondary Accent outline/text color inherited from the active Material 3 theme.
--mad-button-disabled-opacity Opacity percentage used when mixing disabled button colors.
