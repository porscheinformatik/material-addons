Button Components

Selector Appearance Inputs Description
mad-primary-button matButton="filled" type, title, disabled Primary page action. Use once per action area when possible.
mad-outline-button matButton="outlined" type, title, disabled, color Secondary action. The color input supports primary, accent, and warn.
mad-danger-button matButton="filled" type, title, disabled Destructive action using the error color treatment.
mad-link-button matButton="text" type, title, disabled Tertiary text action using the primary color treatment.
mad-icon-button matIconButton type, title, disabled Icon-only action. Provide a title for accessible labeling and tooltip text.

madButton Directive

API Type Default Value Description
madButton Directive - Apply to Angular Material buttons such as matButton="outlined" or matButton="elevated".
color ThemePalette 'primary' Sets Material and Material Addons color classes. Supported values are primary, accent, and warn.
outline boolean true Adds the outline helper class used to keep the outline border in sync with the text color.

Button Group

Selector Content Description
mad-button-group button or a elements using Angular Material button directives Groups related Material buttons and normalizes the shared borders and corner radii.

Events

Event Description
- Button components do not expose custom output events. Use native click events on the host element.
