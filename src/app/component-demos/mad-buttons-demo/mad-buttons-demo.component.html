Import the ButtonModule in your app.module.ts and use one of:
  • mad-primary-button for the primary action on a page, eg. SAVE. There should only be one primary action on one page.
  • mad-outline-button for secondary actions like CANCEL, CLOSE. This button might occur multiple times on a page
  • mad-danger-button for very dangerous actions, that can't be undone.
  • mad-link-button tertiary actions that are not primary and not secondary. like ADD ITEM to a list within a page.
  • mad-icon-button for icon-only buttons. The primary color of the icon shows the user that it's clickable.

A good reference for button usage can be found in the Clarity button styleguide.
