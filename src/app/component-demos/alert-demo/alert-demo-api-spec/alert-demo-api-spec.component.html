Properties

Property Type Default Value Description
type AlertType 'info' Sets the semantic type of the alert. Supported values are success, info, warning, and error.
message string '' Text message displayed inside the alert.
size AlertSize 'medium' Sets the alert density. Supported values are small, medium, and large.
actionText string '' Optional action label. When empty, no action is shown.
closeable boolean true Controls whether the close button is rendered.

Events

Event Description
close Emits when the user activates the close button.
action Emits when the user activates the alert action text.
