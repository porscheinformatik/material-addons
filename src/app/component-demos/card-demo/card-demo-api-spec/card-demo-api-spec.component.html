Properties

Property Type Default Value Description
title string | undefined undefined Optional title displayed in the card header.
readonly boolean true Controls whether edit and footer actions are hidden.
editMode boolean false Controls whether save and cancel actions are shown for a non-readonly card.
expandable boolean true Controls whether the card content can be collapsed and expanded.
expanded boolean true Controls the current expanded state. Supports two-way binding.
editText string 'NOT SET' Tooltip text for the edit action button.
saveText string 'NOT SET' Label and title for the save button shown in edit mode.
saveDisabled boolean false Disables the save button.
cancelText string 'NOT SET' Label and title for the cancel button shown in edit mode.
cancelDisabled boolean false Disables the cancel button.
additionalActionIcon string | undefined undefined Optional Material icon name. When provided, an additional header action button is shown.
additionalActionText string '' Tooltip text for the additional header action.

Events

Event Description
edit Emits when the user activates the edit action. The card expands before the event emits.
save Emits when the user activates the save action in edit mode.
cancel Emits when the user activates the cancel action in edit mode.
additionalAction Emits when the user activates the additional header action.
