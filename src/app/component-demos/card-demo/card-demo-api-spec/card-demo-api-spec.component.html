Properties

Property Type Default Value Description
title string | undefined undefined (Optional) Title displayed in the card header.
readonly boolean true Controls whether the card is in readonly mode.
editMode boolean false When enabled, the card displays footer action buttons and hides the edit button.
expandable boolean true Controls whether the card content can be collapsed and expanded.
expanded boolean true Controls whether the card content is currently visible.
editText string 'NOT SET' Tooltip text for the edit action button.
saveText string 'NOT SET' Label and title for the save button shown in edit mode.
saveDisabled boolean false Disables the save button when set to true.
cancelText string 'NOT SET' Label and title for the cancel button shown in edit mode.
cancelDisabled boolean false Disables the cancel button when set to true.
additionalActionIcon string | undefined undefined (Optional) Material icon name for an additional header action button. When provided, the additional action button is shown.
additionalActionText string '' Tooltip text for the additional action button.

Events

Event Description
edit Emits when the edit button is clicked.
save Emits when the save button is clicked.
cancel Emits when the cancel button is clicked.
additionalAction Emits when the additional action button is clicked.
