Use to display structured content inside a Material card with optional header actions, edit mode, expandable content, and footer actions.

Use When

  • Grouping related content into a clear card layout
  • Displaying readonly or editable form sections
  • Allowing users to expand or collapse a section
  • Providing contextual header actions such as edit or additional actions

Behavior

  • Supports an optional title in the card header
  • Supports expandable and collapsible content
  • Shows an edit button when the card is not readonly and not already in edit mode
  • Shows save and cancel actions when the card is in edit mode
  • Supports an optional additional action button in the header
  • Allows rendering cards with or without a title

Import the CardComponent in your app.module.ts and use it via <mad-card> .
