The Addons extend the default Clarity cards by providing following additional types:

  • Editable card. The Editable card provides an additonal action to trigger an edit action. Also an additionalAction can be defined next to the edit action (eg. for help buttons)
  • Expandable card. The card can be expanded or closed.

Import the CardModule in your app.module.ts and use it via <mad-card>

You can also enable collapse / expand
