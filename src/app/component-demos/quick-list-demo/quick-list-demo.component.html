Use to input an extensible list of input allItems

Use When

  • Extensible list of input types (select boxes, inputs, ...) is needed

Behavior

  • initially one select box is displayed
  • click on "Add option", adds another empty input-row
  • click on "trashbin-icon", removes input-row
    • last item, input mandatory: "trashbin-icon" is hidden
    • minItems defined, the "trashbin-icon" is hidden, when only minimum items available in quick list
    • last item, input optional: input-row is removed, only "ADD OPTION" remains.

Import the QuickListModule in your app.module.ts and use it via <mad-quick-list>.

Basic usage with optional values and no limits


Required value for last name and minimal 2 and maximal 5 entries set.


With<mad-quick-list-compact> there is also an compact version of the quick list. The "add"-Button gets replaced with an "plus-icon" in the last row
