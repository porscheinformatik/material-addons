Properties

Property Type Default Value Description
actionName string '' The tooltip text and accessible label of the action button.
id string '' (Optional) Sets the HTML id attribute on the rendered anchor element.
icon string 'add' The Material icon name displayed inside the action button.
routerLink string '' The router link used for navigation when the action button is clicked.
liftHigher boolean true Moves the floating action button higher to provide additional bottom spacing, for example when a fixed paginator or footer is present.
liftHigher2 boolean false Renders the button as a mini-fab and adjusts its floating position.

Events

Event Description
- This component does not expose custom output events. Navigation is handled through routerLink.
