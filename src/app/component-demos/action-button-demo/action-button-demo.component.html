Use to display a floating Material action button that stays fixed in the lower-right area of the page and navigates to another route.

Use When

  • Providing a prominent primary page action
  • Navigating quickly to create, edit, or secondary action pages
  • Keeping an always-visible floating action available on the screen

Behavior

  • Renders as a floating action button fixed to the lower-right corner
  • Supports a default fab and a mini-fab variant
  • Supports custom icon, tooltip text, id, and router link
  • The liftHigher and liftHigher2 inputs can be used to move the button higher when additional bottom spacing is needed

Import the MaterialActionButtonComponent in your app.module.ts and use it via <mad-material-action-button> .
