Properties

Property Type Default Value Description
label string required The text displayed inside the tile.
variant TileVariant 'info' The visual style of the tile. Possible values are success, info, warning, and error.
size TileSize 'lg' The size of the tile. Possible values are sm, md, and lg.
icon string | null null Optional Material icon name displayed together with the label.
iconPosition TileIconPosition 'start' Controls icon placement relative to the label. Possible values are start and end.

Event Description
— This component does not emit any events.
