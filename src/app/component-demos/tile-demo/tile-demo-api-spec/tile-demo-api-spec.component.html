Properties

Property Type Default Value Description
label string required Text displayed inside the tile.
variant TileVariant 'info' Sets the semantic tile color. Supported values are success, info, warning, and error.
size TileSize 'md' Sets tile density. Supported values are sm, md, and lg.
icon string | null null Optional Material icon name displayed with the label.
iconPosition TileIconPosition 'start' Controls icon placement. Supported values are start and end.

Events

Event Description
— This component does not emit custom events.
