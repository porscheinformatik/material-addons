Use to display prominent tile content with different variants, sizes, and optional icons.

Use When

  • Showing short semantic labels such as statuses, updates, or alerts
  • Highlighting important information in dashboards and overview pages
  • Presenting compact content in a visually prominent tile layout

Behavior

  • Supports multiple tile variants: default, success, info, warning, error
  • Supports multiple sizes: small, medium, large (default is medium)
  • Allows an optional Material icon
  • Supports icon placement at the start or end of the label
  • Wraps longer labels to preserve layout in smaller containers

Import the TileComponent and use it via <mad-tile> .
